# FitProof 证据库收集指引（组员版）

> 目标：为核验功能收集**权威指南层文档**。AI 之后会把这些文档自动拆成可引用的结论条目。
> 你们只负责：找官方文件 → 下载 → 登记。**不需要写代码。**

## 一、收什么（很重要，先读）

✅ **只收这类**："指南层"文件 —— 官方机构发布的膳食指南、临床指南科普版、立场声明、共识、官方科普页。
❌ **不收这类**：单篇研究论文（PubMed 上的期刊文章）、新闻报道、公众号文章、百科词条、任何非官方来源的转载 PDF。

判断口诀：**发布者必须是权威机构本身**（学会/协会/卫健委/WHO），且从**官方网站**下载。

## 二、首批四个领域与推荐来源

每个领域找 2~3 份即可，首批总共 8~10 份。下面是建议的查找方向（以官网实际可下载的为准，找不到某一份就换同机构其他文件）：

### 1. 孕期营养（钩子话题：孕妇能不能吃蛋黄）
- 中国营养学会《中国居民膳食指南（2022）》孕期妇女部分 — 官网 dg.cnsoc.org
- 中国营养学会《孕期妇女膳食指南》
- WHO 孕产妇营养相关 Fact Sheet（who.int，有中文页）

### 2. 婴幼儿与儿童健康（宝妈人群）
- 中国营养学会《婴幼儿喂养指南》
- 国家卫健委《婴幼儿喂养健康教育核心信息》— nhc.gov.cn
- WHO 母乳喂养/辅食添加 Fact Sheet

### 3. 运动健身（已有真实数据的老阵地）
- WHO《关于身体活动和久坐行为的指南》（2020，有中文版）
- ACSM 立场声明（acsm.org，英文可收，登记时备注"英文"）
- 《中国人群身体活动指南（2021）》

### 4. 日常膳食与常见谣言区（鱼油、保健品、胆固醇等）
- 《中国居民膳食指南（2022）》通用部分（一份顶多份，谣言高发区大多被它覆盖）
- 中国营养学会关于膳食补充剂/鱼油类的科学共识
- 国家市场监管总局保健食品消费提示（samr.gov.cn）

## 三、怎么操作（每份文件 3 步）

1. **下载**：从官方网站下载 PDF（或网页版另存为 PDF）。
2. **命名并存放**：文件名用拼音或英文短名，如 `yunqi-shanshi-zhinan-2022.pdf`，放进 `backend/evidence/raw/` 文件夹。
3. **登记**：在下面第四节的表格里加一行（直接编辑本文件，或另建共享表格同步给负责人）。

⚠️ 注意：
- 一定记录**官方链接**（评委会点开验证，链接失效=白收）。
- 版本要最新（指南有年份，收最新版）。
- 拿不准"算不算权威"就先登记、备注存疑，由负责人裁定，**宁可多收再删**。

## 五、后续流程（你们不用做，仅供了解）

收齐后由主开发跑 `ingest_evidence.py`：AI 自动把每份文档拆成结构化结论条目（说法 + 出处章节 + 推荐强度 + 官方链接）→ 人工抽检 10~20% → 入库。之后核验功能的每条判定都会引用到你们收的这些文件上。
